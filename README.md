# Installation
### on Unix
```console:
curl -L https://raw.githubusercontent.com/borley1211/dotfiles/master/etc/install | bash
```

### on Windows
**Please run it in *WSL*.**
```console:
curl -L https://raw.githubusercontent.com/borley1211/dotfiles/master/etc/install-win | bash
```

# Deployment
```console:
$ cd ${DOTPATH}
$ make deploy
```
